CSS3 教程

CSS3边框
border-radius
box-shadow
border-image

CSS3背景
background-size
background-origin(规定背景图片的定位区域，可以content-box,padding-box，border-box)
background-clip (背景从哪里开始被裁剪)

CSS3文本
text-shadow(文字阴影)
word-wrap(单词分割)

CSS3字体
自定义字体
@font-face{
font-family:xxx;
src:url('xxx.ttf'),url('xxx.eot');
}

CSS3 2D转换
transform:
translate(x,y);
rotate(xxdeg);
scale(1.5);
skew();//沿着x，y扭曲
matrix();
transform-origin:50%,50%;//默认几何中心

CSS3 3D转换
transform:
rotateX(xxdeg);
rotateY(xxdeg);
rotateZ(xxdeg);

CSS3 过渡
transition:all 2s linear 0s;

CSS3 动画
定义动画
@keyframes myanima
{
from {}
to {}
}

@keyframes myanima
{
0% {}
25% {}
50% {}
100% {}
}
div{
animation:myanima 2s;
}

CSS3 多列
div{
column-count:3;
column-gap:20px;
column-rule:3px outset #fff;
}

resize规定是否可由用户调整尺寸
